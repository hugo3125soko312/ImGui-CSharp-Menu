## Introduction

This project provides a C# implementation of a graphical user interface (GUI) menu using **ImGui**, a popular immediate mode GUI library. It is designed to be lightweight, easy to integrate, and highly customizable for various C# applications, such as game mods, tools, or utilities.

## Features

- **Immediate Mode GUI**: Leverages the power of ImGui for fast and efficient rendering.
- **Customizable**: Easily modify the appearance and functionality of the menu.
- **Lightweight**: Minimal overhead, making it suitable for performance-critical applications.
- **Cross-Platform**: Works with any C# environment that supports ImGui.
